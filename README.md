# TrendPulse - Data Collection

## Overview
This project collects top trending stories from Hacker News API and categorizes them into different topics.

## Features
- Fetches top stories using API
- Categorizes into:
  - Technology
  - World News
  - Sports
  - Science
  - Entertainment
- Limits 25 stories per category
- Stores structured data in JSON format

## Technologies Used
- Python
- Requests Library
- JSON

## Output
The output is saved in:
data/trends_YYYYMMDD.json
